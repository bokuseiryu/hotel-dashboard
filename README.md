# ホテル業績ダッシュボード

酒店月度经营会议数据可视化分析系统

## 功能特点

- 📊 **可视化展示**：KPI卡片、趋势图表、饼图、柱状图
- ✏️ **数据管理**：在线录入、编辑、删除月度数据
- 💾 **数据持久化**：自动保存到浏览器本地存储
- 📥 **导入/导出**：支持JSON格式数据备份和恢复
- 📱 **响应式设计**：适配电脑和平板设备

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到Netlify

1. 将代码推送到GitHub仓库
2. 在Netlify中连接GitHub仓库
3. 设置构建命令：`npm run build`
4. 设置发布目录：`dist`
5. 点击部署

## 数据说明

数据保存在浏览器的localStorage中，包括：
- 月份
- 売上高（营收）
- 販売客室数（客房数）
- 稼働率
- ADR（平均房价）
- RevPAR
- 客源构成（国内/海外比例）
- 预订渠道数据

## 技术栈

- React 18
- Vite
- TailwindCSS
- Recharts（图表库）
- Lucide React（图标）
